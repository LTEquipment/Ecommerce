# Customer logos

The homepage "Our customers" wall renders one logo per entry in
`lib/customers.ts`, loaded from `/public/customers/<slug>.avif`.

All 38 logos below are already installed (transparent AVIF, white / light artwork
on the dark band). If a file is ever missing, that cell falls back to a clean text
nameplate, so the wall always looks finished. To swap one, replace its `.avif`
here (transparent, ~600px wide, trimmed of surrounding whitespace).
