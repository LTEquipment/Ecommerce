# Security Policy

## Reporting a vulnerability

If you believe you've found a security vulnerability in this project, please report it
**privately** — do not open a public issue.

- **Preferred:** open a private report through GitHub Security Advisories
  (**Security → Report a vulnerability** on this repository).
- **Or** email the L&T team at `customer@ltfse.com` with the details.

Please include the affected page or endpoint, a description of the issue, and a
proof-of-concept or reproduction steps where possible. We'll acknowledge the report and
keep you updated as we work on a fix.

## Scope

This repository powers the L&T Restaurant Equipment storefront. Reports concerning
authentication and sessions, checkout and order handling, admin access, and customer data
exposure are especially appreciated.

## Supported versions

The deployed `main` branch is the only supported version.
